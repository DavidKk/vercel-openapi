# Tasks

Execution layer: what to do now vs later vs done. AI reads **active** to know current work.

| Dir          | Purpose                                                       |
| ------------ | ------------------------------------------------------------- |
| **backlog/** | Tasks not yet started. Move here when defined but not active. |
| **active/**  | Tasks in progress. Primary entry: `active/current.md`.        |
| **done/**    | Completed tasks. Move here when a task is finished.           |

When a task is completed: move its entry to `done/` or append to `done/README.md`, then remove from `active/`.
